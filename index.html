Akan Name Application

Program Description

This web application takes the user's date of birth and gender; then calculates the day of the week, they were born in. The application then uses this input to assign the user their Akan Name.
The Akan naming convention is a Ghanaian cultural event that takes place among the Akan people of Ghana. The name given possesses a deeper cultural meaning and is said to influence the person's character and soulfulness.
